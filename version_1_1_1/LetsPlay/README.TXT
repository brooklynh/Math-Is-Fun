=====Let's Play!==========
created by Brooklyn Hofer.
==========================

How to install and play:
	1. Update the registry with student information.
	2. Install/update Java.
	3. Run the executable jar file.

How to access scores:
	"LetsPlay/Students" folder has game information on each student.

Math Is Fun:
	Solve randomly generated math problems.
	4 game modes: Addition, Subtraction, Division, Multiplication
	3 difficulties: Easy, Medium, Hard
	Division lets user choose precision for answers.
